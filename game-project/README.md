A Simon Says memory game built using HTML, CSS, and JavaScript.

## About the Project
This is a classic Simon Says game where the player must remember and repeat
an increasing sequence of colors. Each correct round advances the level.

## How to Play
- Press any key to start the game
- Watch the color sequence carefully
- Repeat the sequence by clicking the buttons
- Each level adds one new color
- A wrong click ends the game

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)

## Features
- Random color sequence generation
- Level-based progression
- Visual feedback for game and user actions
- Keyboard and mouse interaction

Beginner-friendly JavaScript mini project
